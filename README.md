### Prerequisites

### Installation

```bash
# Clone the repository
git clone https://github.com/LyleListon/MayArbi.git
cd MayArbi

# Install DexMind dependencies
cd dexmind
npm install
npm run build

# Install other MCP servers
cd ../filesystem-mcp-server
npm install
npm run build

npm install
npm run build
```
